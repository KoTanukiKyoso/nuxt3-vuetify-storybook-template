# Nuxt3 with Vuetify and Storybook Template

Nuxt3とVuetifyを使用していて、Storybookを使用したい方のためのテンプレート。

Bunでテストしてますがnpm等でも問題ないと思います。

## バージョン

Nuxt3: 3.13.0  
Vuetify: 3.7.1  
Storybook: 8.2.9

## インストール

```bash
npm install
```

など

## 実行

### Storybook
```bash
npm run storybook
```

### 開発
```bash
npm run dev
```

### Storyファイル自動生成（Bash、Git Bash等）
対象のvueファイルを含む（deep）フォルダで「create-story.bash」を実行
